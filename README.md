Não tinha uma API funcional de ben 10 entao tivemos que criar uma API do zero, Os dados dos aliens são achados localmente no aplicativo.

 Sobre o Projeto
O Omnitrix Database é um aplicativo mobile desenvolvido em React Native que funciona como uma enciclopédia digital dos aliens,ominitrix e series do universo Ben 10.

 Funcionalidades Principais
 Catálogo Completo: Lista organizada de todos os aliens disponíveis

 Filtros por Série: Filtre aliens por série (Clássico, Alien Force, etc.)

 Estatísticas Detalhadas: Visualize atributos como força, inteligência, velocidade, durabilidade e versatilidade

 Expansão de Cards: Toque para expandir e ver informações completas

 Interface Intuitiva: Design inspirado no tema Ben 10 com cores verdes características

 Conteúdo Incluído
6 Aliens Iniciais: Heatblast, Four Arms, XLR8, Diamondhead, Swampfire e Humungousaur

Informações Detalhadas:

Espécie e planeta natal

Habilidades e custos de poder

Estatísticas em barras visuais

Curiosidades e triviais

Primeira aparição

Fraquezas e transformações

 Tecnologias Utilizadas
React Native com TypeScript

Expo Vector Icons (Ionicons)

Componentes Nativos: ScrollView, StyleSheet

Estado com React Hooks (useState)

 Design System
Cores Principais: Verde Omnitrix (#27AE60)

Layout: Cards expansíveis com informações hierárquicas

Ícones: Sistema de ícones representando cada alien

Responsivo: Adaptado para dispositivos móveis
